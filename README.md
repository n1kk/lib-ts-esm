# npm ts library boilerplate

- `typescript`
- `es module`
- `jest`
- `prettier`
- `eslint`
- `husky` + `lint-staged`
- `pnpm`: recommended
  - entry point override for publishing, for other packagers update the `exports` field
  - access: public

```bash
degit n1kk/lib-ts-esm
pnpx degit n1kk/lib-ts-esm
npx degit n1kk/lib-ts-esm
```
